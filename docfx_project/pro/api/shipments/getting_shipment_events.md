# Getting Shipment Events

Get Tracking Events

`GET https://api.sorted.com/pro/tracking/{shipment_reference}`

Get Tracking Events by Custom Reference

`GET https://api.sorted.com/pro/tracking/custom_reference/{shipment_reference}`
